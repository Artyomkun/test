# Отчеты о покрытии кода тестами

В этом каталоге хранятся отчеты о покрытии кода, сгенерированные Jest и Istanbul.

## Что такое покрытие кода?

Покрытие кода (code coverage) - это метрика, которая показывает, какая часть кода была выполнена во время тестов. 

### Основные метрики:

1. **Statements (Операторы)** - процент выполненных операторов кода
2. **Branches (Ветви)** - процент покрытых ветвей условий (if/else, switch)
3. **Functions (Функции)** - процент вызванных функций/методов
4. **Lines (Строки)** - процент выполненных строк кода

## Файлы в каталоге

- `coverage-summary.json` - сводная статистика в JSON формате
- `lcov.info` - данные покрытия в LCOV формате
- `clover.xml` - отчет в Clover формате (для SonarQube)
- `index.html` - главный HTML отчет
- `base.css`, `block-navigation.js`, `prettify.css`, и т.д. - вспомогательные файлы для HTML отчета
